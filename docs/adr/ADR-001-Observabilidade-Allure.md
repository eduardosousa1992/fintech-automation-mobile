# ADR 001: Implementação de Observabilidade com Allure Report
## Status: ACEITO (18/03/2026)
## Contexto: Necessidade de evidências visuais e rastreabilidade para auditoria em Fintech.
## Decisão: Uso de Allure Report com injeção de metadados e screenshots por passo.
